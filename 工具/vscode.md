## 常用快捷键

| 快捷键      | 意义                 |
| ----------- | -------------------- |
| ⌘K M        | change file language |
| ⌘K V        | preview markdown     |
| alt shift U | 上传 vscode 配置     |
| alt shift D | 下载 vscode 配置     |

### JavaScript Code Snippets 代码片段

| 快捷键 | 意义                                        |
| ------ | ------------------------------------------- |
| clg    | console.log()                               |
| sto    | setTimeOut                                  |
| prom   | return new Promise((resolve, reject) => {}) |
| thenc  | .then(res => {}).catch(err => {})           |
| fof    | for...of                                    |
| fin    | for...in                                    |
| dob    | 解构赋值                                    |
| sto    | setTimeout                                  |
